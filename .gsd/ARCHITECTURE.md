CyberGuard is a multi-page Flask web application that accepts a YouTube video URL, fetches its public comments via the YouTube Data API v3, classifies each comment across 6 toxicity categories using a fine-tuned ToxicBERT model, and renders a visual analytics dashboard. Results are persisted in SQLite for history access. The app is deployed publicly via a cloud host (e.g. Render/Railway).

## Data Flow

1. User submits YouTube URL via POST `/analyze`
2. Flask extracts `video_id` from URL
3. YouTube API fetches up to 2000 comments (paginated)
4. Each comment вҶ’ ToxicBERT tokenizer вҶ’ model forward pass вҶ’ sigmoid вҶ’ 6 scores
5. Scores aggregated via pandas вҶ’ stats computed
6. Results saved to SQLite `analyses` table
7. Flask renders `dashboard.html` with chart data, stat cards, comment tables
8. User can flag/hide comments (client-side JS) or download report
