# Plan 9.3: Global Mouse & Scroll Parallax Effect

## Objective
Implement global javascript logic to inject floating 3D elements (shapes or characters) into the background of all pages that respond to mouse movement and scrolling physics. 

## Context
- static/js/main.js
- templates/base.html
- static/css/style.css

## Tasks

<task type="auto">
  <name>Add Parallax CSS & Elements</name>
  <files>
    - templates/base.html
    - static/css/style.css
  </files>
  <action>
    - Insert container `div`s with absolute positioning in `base.html` containing floating aesthetic DOM elements (e.g. geometric shapes, dots, or blurred SVGs) with the `.parallax-element` class.
    - Add transition styling to `.parallax-element` in `style.css` so movement is smooth (transition-out cubic-bezier etc).
  </action>
  <verify>DOM logic exists in base template globally</verify>
  <done>HTML structure is ready for JS bridging</done>
</task>

<task type="auto">
  <name>Add Parallax JS Logic</name>
  <files>
    - static/js/main.js
  </files>
  <action>
    - Bind `mousemove` event listeners to calculate `e.clientX / window.innerWidth` offsets.
    - Bind `scroll` event listeners.
    - Apply dynamic `transform: translate3d()` to `.parallax-element` nodes based on calculated offset to create a faux-3D depth effect.
  </action>
  <verify>Validate interactions locally</verify>
  <done>Global 3D elements shift smoothly based on user cursor and scroll actions</done>
</task>

## Success Criteria
- [ ] Background visuals shift on mouse movement 
- [ ] Background visuals shift during page scrolling
- [ ] Effect is applied evenly to all pages via `base.html`
