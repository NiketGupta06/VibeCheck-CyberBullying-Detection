Problem Description -
Cyberbullying on social media platforms causes psychological harm and violates user safety. Manual monitoring is inefficient due to the large volume of user-generated content. Gain experience in secure and ethical data handling
Proposed Solution
A web-based application that analyzes social media text and flags cyberbullying content, along with a trained ML model and project documentation.
Technology Stack
Python, NLP, Machine Learning, Flask
